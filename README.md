## 実行環境
- python 3.9.12
- tcl-tk 8.6.12
  - Mac ユーザーの場合
    - brew をインストール
    - tcl-tk のバージョン確認
        ```
        brew info tcl-tk
        ```
    - バージョンが 8.6 以前の場合
      - ```brew upgrade tcl-tk```
<br>
<br>

## 実行方法
- pipenv を使う場合
  - 環境の再現
    - pipenv update
- 自前の python 環境を使う場合
- docker を使う場合
